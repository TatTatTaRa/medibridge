# 정형외과·재활 병원정보시스템(HIS) 시스템 정의

정형외과·재활의학과 특화 HIS 프로젝트의 **시스템 구성요소별 개념·목적·기능·프로세스 흐름**을 정리한 문서 세트입니다.
기획보고서가 "프로젝트 관점"이라면, 이 문서들은 **시스템 구성요소(HIS·OCS·EMR·PACS …) 관점**으로 분류한 기술 참고 문서입니다.

> 모든 흐름도는 GitHub에서 자동 렌더링되는 **Mermaid**로 작성했습니다. (Preview 탭에서 그림으로 보입니다)

## 전체 구성 한눈에 보기

```mermaid
flowchart TD
    P["환자"] --> WON["원무 / 접수·수납"]
    WON --> EMR["EMR<br/>전자의무기록"]
    EMR --> OCS["OCS<br/>처방·오더 전달"]
    OCS --> LIS["LIS<br/>진단검사"]
    OCS --> RIS["RIS / PACS<br/>영상검사"]
    OCS --> PHARM["약/주사/처치"]
    OCS --> REHAB["재활 특화 모듈 ★"]
    LIS --> EMR
    RIS --> EMR
    REHAB --> PORTAL["환자 포털 / PHR ★"]
    EMR --> CLAIM["수납·청구"]
    PORTAL --> P
    subgraph 공통기반
      STD["표준·연계<br/>KCD·DICOM·HL7"]
      SEC["보안·권한·로그"]
    end
```
